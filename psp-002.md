## System
### system_name
Get the node's implementation name.

**Parameters**

None

**Returns**

`STRING` - The node's name.

**Example**

```bash

curl -H "Content-Type: application/json" -d '{"id":1, "jsonrpc":"2.0", "method": "system_name", "params":[]}'

// Response
```
